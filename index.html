Compressed Data

Compressed data eller Datakompression som vi säger på svenska är när data omkodas så att filen tar upp mindre utrymme på din disk. Ett exempel på detta är när man tex laddar ner ett spel så tar det ofta upp vädigt mycket plats på din dator men då när färdigt så kan det ofta kompimeras och ta mindre plats på datorn.

CDN

CDN är ett servernätverk som levererar information. Detta hjälper din webbplats att fungera snabbt och smidig

JQuery

jQuery är världens mest populära JavaScript-bibliotek. Det är bland annat designat för att göra det enklare att anropa och modifiera element på tex en hemsida.

Flexbox

Flexbox även kallat Flexible Box Layout är en typ av layout på en hemsida som anpassar sig efter skärmstorleken.

Grid

En grid-layout gör så att man delar in sin container i rutnät, där man bestämmer hur många rader och kolumner som rutnätet ska bestå av.

Local Bootstrap

Det som skiljer Local Bootstrap och det vanliga bootstrap är att de lokala är mer användbart för heteroscedastic(vet inte vad det är på svenska) data.

HTTP

HyperText Transfer Protocol eller förkortat HTTP är ett så kallat kommunikationsprotokoll och används för att ladda hemsidor på internet via WWW. Utan HTTP så hade din enhet inte kunnat kommunicera med servrar vilket hade ledit till att din webbläsare inte hade kunnat läsa innehållet från olika hemsidor.

HTTPS

HTTPS är ett sorts protokoll som används för krypterad transport av data. Det kan förklaras som det som sker mellan din dator och en hemisda är krypterad, alltså hemlig.

Code Snippets

Code Snippets eller klipp, snutt som det kan översättas till är ett kort stycke eller programkod i detta fallet som man klippt ut ifrån en större helhet. Inom webb används detta ofta för att spara en kort programkod som kan/ska användas flera gånger. Jag använde detta vid "card" sektionen då jag inte orkade skriva om koden sex gånger.
